map  :

1 : Cration d'une clé google map pour avoir acces a l'api
2 : restreindre la clé pour la rendre fonctionnelle
3 : (optionel) ajouter le paiement, donc une carte bancaire pour pouvoir voir la carte sans le mode developpeur

4 : Ajout de 4 fichier 


Permier fichier -> une vue map_carte accesible en public 
Deuxieme fichier -> un modele pour pouvoir utiliser des requetes
troisieme fichier -> un controleur qui recupere l'id, la latitude et la longitude du velo
quatrieme fichier -> Un fichier Javascript qui nous permet d'améliorer la carte et de collecter les données 
Il crée un Json qui envoie les donnée dans un XML ses meme donnée qui sont exploité dans le modele.
